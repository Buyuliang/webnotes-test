# cmd_shell

在 ranger 里执行 shell 命令可以这样做：

  1. 临时执行一条命令：`!`

  • 按 `!`
  • 输入命令，回车执行
  • 执行完会提示按任意键回到 ranger


  2. 在当前目录开一个子 shell：`S`（大写）

  • 按 `S`
  • 会在当前目录打开一个 shell，可连续输入多条命令
  • 输入 `exit` 或按 Ctrl+D 关掉 shell，回到 ranger


  3. 用 ranger 命令行：`:`

  • 按 `:` 进入命令行
  • 输入 `shell` 或 `terminal`（视版本而定）可打开 shell
  • 部分版本也支持 `:! 命令` 这种写法


  4. 把当前文件/路径传给命令

  • 先选中文件，再按 `!`
  • 输入命令时可用 `%s` 表示当前选中的文件，`%d` 表示当前目录

    例如：
        ! xdg-open %s 用默认程序打开选中文件
  常用：
  • `!` → 快速执行一条命令
  • `S` → 在当前目录开 shell，方便连续操作
