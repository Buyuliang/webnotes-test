# 本地调试 - 无需安装任何工具

## 方法一：直接打开 HTML 文件（最简单，但功能受限）

1. 直接双击 `index.html` 文件
2. 会在默认浏览器中打开

**注意**：某些功能可能受限（如 CORS），但基本功能可以测试。

## 方法二：使用 VS Code Live Server（推荐）

### 安装步骤：
1. 打开 VS Code
2. 安装扩展：搜索 "Live Server"（作者：Ritwick Dey）
3. 右键点击 `index.html` 文件
4. 选择 "Open with Live Server"
5. 浏览器会自动打开 `http://127.0.0.1:5500`

**优点**：
- 无需安装 Python 或 Node.js
- 自动刷新
- 支持热重载

## 方法三：使用 Python（如果已安装）

### 检查是否已安装 Python：
```bash
python --version
```

### 如果已安装，运行：
```bash
cd g:\webnotes-test
python -m http.server 8000
```

然后访问：`http://localhost:8000`

### 如果没有安装 Python：
1. 下载：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 重新运行上面的命令

## 方法四：使用 Windows 内置功能（PowerShell）

在 PowerShell 中运行：

```powershell
cd g:\webnotes-test
$listener = New-Object System.Net.HttpListener
$listener.Prefixes.Add("http://localhost:8000/")
$listener.Start()
Write-Host "服务器已启动: http://localhost:8000"
Write-Host "按 Ctrl+C 停止"

while ($listener.IsListening) {
    $context = $listener.GetContext()
    $request = $context.Request
    $response = $context.Response
    
    $localPath = $request.Url.LocalPath
    if ($localPath -eq "/") { $localPath = "/index.html" }
    
    $filePath = Join-Path $PWD $localPath.TrimStart('/')
    
    if (Test-Path $filePath) {
        $content = [System.IO.File]::ReadAllBytes($filePath)
        $response.ContentLength64 = $content.Length
        $response.OutputStream.Write($content, 0, $content.Length)
    } else {
        $response.StatusCode = 404
    }
    
    $response.Close()
}
```

## 常见问题

### Q: 为什么不能直接打开 HTML 文件？
A: 某些浏览器功能（如 fetch API、localStorage）在 `file://` 协议下可能受限。使用 HTTP 服务器可以避免这些问题。

### Q: 没有 Python 也没有 Node.js 怎么办？
A: 使用 VS Code Live Server 扩展，这是最简单的方法。

### Q: 如何检查端口是否被占用？
A: 
```bash
# Windows
netstat -ano | findstr :8000

# 如果被占用，可以换其他端口
python -m http.server 8080
```
