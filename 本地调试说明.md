# 本地调试说明

## ⚡ 快速开始（推荐）

### 最简单：使用 VS Code Live Server
1. 安装 VS Code
2. 安装 "Live Server" 扩展
3. 右键点击 `index.html` → "Open with Live Server"
4. 完成！

### 或者：双击启动脚本
- **Windows**: 双击 `start-server.bat`
- **Linux/Mac**: 运行 `./start-server.sh`

---

## 方法一：使用 Python HTTP 服务器（推荐，最简单）

### Python 3
```bash
# 在项目目录下运行
python -m http.server 8000

# 或者
python3 -m http.server 8000
```

### Python 2
```bash
python -m SimpleHTTPServer 8000
```

然后在浏览器访问：`http://localhost:8000`

## 方法二：使用 Node.js http-server

### 安装
```bash
npm install -g http-server
```

### 运行
```bash
# 在项目目录下运行
http-server -p 8000

# 或者指定端口和允许 CORS
http-server -p 8000 -c-1 --cors
```

然后在浏览器访问：`http://localhost:8000`

## 方法三：使用 VS Code Live Server 扩展

1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html` 文件
3. 选择 "Open with Live Server"
4. 浏览器会自动打开

## 方法四：使用 PHP 内置服务器

```bash
php -S localhost:8000
```

## 方法五：使用 npx serve（无需安装）

```bash
npx serve -p 8000
```

## 方法六：直接打开文件（不推荐）

直接双击 `index.html` 文件，但有以下限制：
- 某些浏览器功能可能受限（CORS 限制）
- 本地存储可能有问题
- 某些 API 可能无法正常工作

## 推荐配置

### 使用 Python（最简单）

1. 打开终端/命令行
2. 进入项目目录：
   ```bash
   cd g:\webnotes-test
   ```
3. 启动服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器访问：`http://localhost:8000`

### 停止服务器
按 `Ctrl+C` 停止服务器

## 调试技巧

### 1. 打开浏览器开发者工具
- Chrome/Edge: 按 `F12` 或 `Ctrl+Shift+I`
- Firefox: 按 `F12` 或 `Ctrl+Shift+I`
- Safari: 按 `Cmd+Option+I` (Mac)

### 2. 查看控制台错误
- 打开 Console 标签页
- 查看 JavaScript 错误和警告

### 3. 网络请求调试
- 打开 Network 标签页
- 查看 API 请求和响应
- 检查是否有 404 或其他错误

### 4. 本地存储调试
- 打开 Application/Storage 标签页
- 查看 localStorage 中的数据
- 可以手动修改或清除数据

### 5. 断点调试
- 在代码中添加 `debugger;` 语句
- 或使用 Sources 标签页设置断点

## 常见问题

### CORS 错误
如果遇到 CORS 错误，确保：
- 使用 HTTP 服务器而不是直接打开文件
- 如果使用 http-server，添加 `--cors` 参数

### 端口被占用
如果 8000 端口被占用，可以换其他端口：
```bash
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### GitHub API 限制
- 确保已配置 GitHub Token
- 检查 Token 是否有正确的权限
- 注意 API 速率限制

## 快速启动脚本

### Windows (start-server.bat)
```batch
@echo off
echo 启动本地服务器...
echo 访问 http://localhost:8000
python -m http.server 8000
pause
```

### Linux/Mac (start-server.sh)
```bash
#!/bin/bash
echo "启动本地服务器..."
echo "访问 http://localhost:8000"
python3 -m http.server 8000
```
